# Managment-de-Tareas

# Libs:
- matplotlib (para mostrar estadistica)
- pick (para el menu)

# Estructura del Programa:

- **Manejar mi Perfil**
  - Ver mi perfil
  - Cambiar nombre
  - Ver mis tareas
  - Cerrar sesión

- **Manejar Personas**
  - Borrar una persona del sistema
  - Manejar solicitudes de roles

- **Manejar Equipos**
  - Agregar nuevo equipo
  - Manejar un equipo
    - Borrar equipo
    - Cambiar nombre del equipo
    - Agregar una persona a un equipo
    - Eliminar una persona del equipo
    - Mostrar estadística
  - Ver equipos
  - Ver equipos más efectivos

- **Manejar Tareas**
  - Agregar tarea
  - Manejar tarea
    - Cambiar datos de tarea
    - Borrar tarea
    - Asignar tarea a un equipo
  - Ver tareas
